# NodeJS Realtime Chat App. <a name="about-project"></a>

A scalable, realtime chat app with Node JS, React JS, and ChatEngine.io.
This app supports group chats, 1-on-1 chats, realtime messaging, image and file support, read receipts and more!

In order to run this project you need:

```sh
 npm install
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
```

Install this project with:

```sh
  cd my-folder
  npm install
```

To run the project, execute the following command:

```sh

  cd backend
  npm run start
```
